# Projet-Avalanches
Analyse des avalanches en France 2000-2023 - base de données SQL + analyses statistique en R.

## Contexte
Projet universitaire réalisé dans le cadre des UE Bases de Données 
et Sciences des Données 2 (Université Paul Valéry, Montpellier 3).

## Objectif
Identifier où se déclarent les avalanches les plus dangereuses 
et quelles stations de ski en sont les plus proches.

## Données
- **ANENA** : 1690 avalanches et 2773 victimes recensées (2000-2023)
- **Station-de-ski.net** : 232 stations de ski françaises

## Outils
- SQL / phpMyAdmin - modélisation et requêtes (MCD, MOD, 4 tables)
- R / RStudio - analyse exploratoire, tests statistiques (khi², ANOVA, corrélation)

## Résultats clés
- 92,7% des avalanches se produisent dans les Alpes
- 78% sont d'origine humaine (randonnée, hors-piste)
- 90% des victimes avec la tête ensevelie décèdent
- Stations les plus touchées : Chamonix, Tignes, Val d'Isère
